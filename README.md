# Plan maestro — 4000 ejercicios de ingeniería en Python

Este repositorio contiene un programa progresivo de **4000 ejercicios** orientado a backend y diseño de software.

## Principios
- No se incluyen soluciones, código, pseudocódigo ni implementaciones.
- Cada ejercicio fuerza razonamiento, descomposición y toma de decisiones técnicas.
- Dificultad progresiva en 6 niveles por módulo.
